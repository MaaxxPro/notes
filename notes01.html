Web Basics

IP: Internet Protocol, the specific address of a computer that identifies it among the billions connected to the web

Domain Names are aliases for IP addresses

Internet =/= Web

Internet is technical infrastructure

Web is a service built on top of that infrastructure

Intranets are private networks, extranets are networks opened up to other networks to share information between.

Both run on the same infrastructure as the web, and so can be accessed from various locations.

Web Page

A document which can be displayed in a web browser such as Firefox, Google Chrome, Opera, Microsoft Edge, or Apple Safari. These are also often called just "pages."

Web Site

A collection of web pages which are grouped together and usually connected together in various ways. Often called a "web site" or a "site."

Web Server

A computer that hosts a website on the Internet.

Search Engine

A web service that helps you find other web pages, such as Google, Bing, Yahoo, or DuckDuckGo. Search engines are normally accessed through a web browser (e.g. you can perform search engine searches directly in the address bar of Firefox, Chrome, etc.) or through a web page (e.g. bing.com or duckduckgo.com).

Browser vs Search Engine

A browser is a piece of software that retrieves and displays web pages; a search engine is a website that helps people find web pages from other websites.

domain names are read right to left

Top Level Domain (TLD)

(“.com” “.net”)

some TLDs enforce stricter policies so it is clearer what their purpose is. For example:

Local TLDs such as .us, .fr, or .se can require the service to be provided in a given language or hosted in a certain country — they are supposed to indicate a resource in a particular language or country.

TLDs containing .gov are only allowed to be used by government departments.

The .edu TLD is only for use by educational and academic institutions.

The Label

a case-insensitive character sequence anywhere from one to sixty-three characters in length, containing only the letters A through Z, digits 0 through 9, and the - character (which may not be the first or last character in the label)

You can create subdomains by adding additional labels, with a dot. “subdomain.label.tld”

Use whois command on domain registry services to query domain name availability

Protocols

Define how data should travel across the web

TCP "Transmission Control Protocol

IP "Internet Protocol"

HTTP "Hypertext Transfer Protocol": The language used by client and server

Component Files, Code Files, and Assets

Component Files make up a website. Code Files are the actual code, and Assets are the rest (images, pdfs, whatever)

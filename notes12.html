Inspecting and Debugging HTML and CSS

Odin Project Link

June 19 2023

Access the inspector in Chrome by pressing f12 or right clicking the page and selecting Inspector.

Elements

Hovering over an element in the elements panel will highlight it on the page.

Clicking the arrow and outline icon in the top left corner will allow cause the elements panel to highlight any elements when you hover the cursor over them on the page.

Styles

Selecting an element in the Elements Panel will cause the Styles panel to display all relevant styles being applied to that element, as well as any overwritten styles.

Styles can be added, removed or edited in the panel and changes will display on the page.

DevTools

ctrl shift p brings up searchable commands for the settings.

"Dock" refers to attached display of DevTools. Dock left will put at left of screen. Undock will detach, can be placed in separate window, second monitor.

Ctrl shift c to acccess elements panel

ctrl shift j to access console and run/debug javascript
